# .netcore-aspnet

Keywords:
* .NET Core
* Kestrel
* ASP.Net Application
* Bower
* Yeoman

```bash
$ npm install -g yo bower
$ npm install -g generator-aspnet
$ cd SampleWebApplication
$ dotnet build
$ dotnet ef database update
$ dotnet run
```

If you get a permission error, run the command under sudo.
